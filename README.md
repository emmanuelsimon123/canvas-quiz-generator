# 📝 Canvas Quiz Generator

An AI-powered tool that turns images of study material into Canvas-compatible quiz files. Upload a photo of your notes, textbook, or worksheet, let the AI generate quiz questions, and export a QTI `.zip` file you can import directly into Canvas LMS.

## Features

- **Multiple question types** — Multiple Choice, True/False, Fill in the Blank, Multiple Answers, Numerical
- **Smart generation modes**:
  - 🤖 **Auto Mix** — AI picks the best variety of question types
  - 🔘 **One Type** — All questions are the same type
  - 🎯 **Pick & Choose** — You select which types to include
- **Graph generation** — Automatically creates matplotlib graphs for algebra and pre-calc questions
- **Canvas-ready export** — Outputs a QTI 1.2 `.zip` file that imports directly into Canvas

## Tech Stack

- [Streamlit](https://streamlit.io/) — Web UI
- [Groq API](https://groq.com/) with **Llama 4 Scout** vision model — AI question generation
- [matplotlib](https://matplotlib.org/) — Graph rendering
- [QTI 1.2](https://www.imsglobal.org/question/index.html) — Canvas-compatible quiz export format

## Setup

1. **Clone the repo**
   ```bash
   git clone https://github.com/emmanuelsimon123/canvas-quiz-generator.git
   cd canvas-quiz-generator
   ```

2. **Install dependencies**
   ```bash
   pip install -r requirements.txt
   ```

3. **Configure your API key**
   ```bash
   cp .env.example .env
   ```
   Then open `.env` and replace `your_groq_api_key_here` with your actual [Groq API key](https://console.groq.com/).

4. **Run the app**
   ```bash
   streamlit run app.py
   ```

## How to Import into Canvas

1. Download the `.zip` file generated by the app.
2. In Canvas, go to your **Course → Settings → Import Course Content**.
3. Select **QTI .zip file** as the content type.
4. Upload the `.zip` file and click **Import**.

## Supported Question Types

| Type | Canvas Type |
|---|---|
| Multiple Choice | `multiple_choice_question` |
| True / False | `true_false_question` |
| Fill in the Blank | `short_answer_question` |
| Multiple Answers | `multiple_answers_question` |
| Numerical Answer | `numerical_question` |
